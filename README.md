# Temperature Humidity DBMS Project
Project Temperature and humidity base on ESP8266, DHT22, firebase, MySQL

## ESP8266
**อุปกรณ์**
1. บอร์ด ESP-8266
2. เซ็นเซอร์ DHT22
3. สายไฟ
4. มิเตอร์แบบสั่น

**Library**
1. ESP8266WiFi.h
2. DHT.h
3. ESP8266WiFiMulti.h
4. ESP8266HTTPClient.h
